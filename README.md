## Description

Backend server for Web Planning Poker.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# start database
$ docker compose up -d

# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
